# Email-Generator
Generate email code from template. Generates C# code from a given email template

Accepted mail format for smart mode

To: to@cpp.com <br/>
From: from@cpp.com <br/>
cc: cc@cpp.com <br/>
bcc: bcc@cpp.com <br/>
attachment: file <br/>
subject: Hello <br/>
Dear X <br/>
I am missing you. <br/>
Yours <br/>
Y <br/>
